# Support

## Where to ask questions
- **GitHub Discussions**: best for general questions, dataset comparisons, and guidance on using the dataset template.

## When to open an Issue
Open an Issue when there is a concrete problem we can fix, such as:
- incorrect or inconsistent metadata/stats in a dataset repository
- broken links in `DOWNLOAD.md` / citation files
- packaging/validation problems related to the dataset template workflow

## What to include
- Link to the dataset repository
- Links to the relevant files (e.g., `METADATA.json`, `SUMMARY.md`, `DOWNLOAD.md`)
- What you expected vs what you observed (screenshots help)

## Security
Please do not post security reports publicly. See `SECURITY.md`.
